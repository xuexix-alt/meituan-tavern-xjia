# 店铺解析测试案例

## 测试案例1：包含[推荐标签]的文本
```
[手机界面开始]
[推荐标签]

一线女星
资本女王
顶流献身
[店铺]
shoptags:
娱乐圈顶流
高情商狐狸
对赌协议
shop_id:20240728103001
name: 【顶流蜜糖】女王的游戏
[套餐]
name: 狐狸的契约·赌上事业的内射
price: 988
[/套餐]
[/店铺]
[手机界面结束]
```

**期望结果**：
- 店铺名称：`【顶流蜜糖】女王的游戏`
- 套餐名称：`狐狸的契约·赌上事业的内射`
